# Bar Banners

## About
This project features visual banners for a bar, created entirely from scratch. The main goal was to design visually appealing banners that reflect the bar’s style and atmosphere.

## Design Focus
- Eye-catching layouts
- Bold and readable typography
- Balanced and appealing color schemes
- Clear visual hierarchy
